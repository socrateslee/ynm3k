# ynm3k(要你命3000)

[ynm3k(要你命3000)](https://github.com/socrateslee/ynm3k)是一个基于Python开发，以辅助调试为主要目的，其他各种莫名其妙的功能都有一些的http服务器，主要功能包括：

- 直接通过json进行配置
- 编写和服务mock数据
- 提供反向代理
- 静态文件服务器（包括把zip文件作为静态目录提供服务）
